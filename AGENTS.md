# AGENTS

Instructions for AI systems working in this repository.

## Mission

Build a local, founder-first automation assistant from scratch.

## Product constraints

1. Local-first by default
2. Human approval for high-impact actions
3. Explainable runs (state, logs, decisions)
4. Small, reversible increments only

## Required workflow

Always follow:

1. PRD
2. Review
3. Tests
4. Implementation
5. QA
6. Ship

Do not skip stages.

## Scope rules

1. One PRD = one smallest valuable update
2. Every PRD must include:
   - design artifact in `docs/pm/designs/`
   - flowchart artifact in `docs/pm/flowcharts/`
3. Do not execute future stages unless requested

## Versioning

1. Keep versions `< 1.0.0` until product is fully functioning
2. Use `0.x.y`

## Source of truth

1. Vision: `docs/pm/VISION.md`
2. PM process and templates: `docs/pm/`
3. Technical specs and diagrams: `docs/tech/`

## Output style

Be concise. Prefer dense context over long prose.
